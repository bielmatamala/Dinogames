# 🦖 Dino Game


Una recreació en **Python** del clàssic **Dino Game de Google Chrome**. En aquest joc controles un petit **T-Rex** que ha de saltar obstacles i sobreviure el màxim temps possible mentre la velocitat del joc augmenta progressivament.

Aquest projecte està pensat com a pràctica de **programació amb Python** i **desenvolupament bàsic de videojocs**.
